# Hospital Appointment (demo project)

Небольшой **рабочий** проект для презентации: запись пациентов к врачам (Flask + SQLite).

## Что умеет
- Просмотр списка врачей и свободных слотов
- Бронирование слота (пациент вводит имя/телефон)
- Просмотр всех записей (простая “админ” страница без логина)
- Отмена записи по ссылке

## Быстрый старт (Windows / PowerShell)
Из папки `hospital-appointment`:

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
py app.py init-db
py app.py run
```

Открой в браузере: `http://127.0.0.1:5000`

## Команды
- `py app.py init-db` — создаёт `instance/app.db` и наполняет демо-данными
- `py app.py run` — запускает сервер

## Идея для презентации (3–5 минут)
- Показать главную страницу: врачи + слоты
- Забронировать слот и увидеть подтверждение
- Открыть страницу “All appointments” и показать запись
- Отменить запись по ссылке (показать, что слот снова свободен)
